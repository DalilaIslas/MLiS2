# Policy Gradient Examples

This project is written in Julia. Go to their [website](https://julialang.org/) to download Julia to run these examples.

Once downloaded, open the folder in the terminal and run:
```bash
julia --project
```
Once inside, run the following commands:
```
using Pkg;
Pkg.activate(".");
Pkg.instantiate();
```
These commands will download all the packages you need to run these examples.
